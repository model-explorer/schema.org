How-to


  • View these instructions later?
    Click on the HELP item in the top menu
  • Add the neighbours to current visualiation?
    Double-click on a node; can be repeated as nodes are added in batches based on the expansion limit
  • Add the neighbours of all nodes of the same type? Double-click on a node while holding down the SHIFT key; all displayed nodes of the same type will be expanded
  • Undo the last expansion (adding of neighbours)?
    Press Ctrl+Z to remove all nodes added in most recent expansion
  • Start a new visualisation with the node and its neighbours?
    Double-click on a node with CTRL key down
  • Delete selected node or edge?
    Press the DEL key
  • Delete all nodes with object type of selected node?
    Press the DEL key while holding down the SHIFT key; all displayed nodes of the same type will be deleted
  • Show description and other properties in this box?
    Select the node with a single click
  • Lock a position of a certain node?
    Drag and drop the node
  • Fix position of all nodes?
    Press "F" key to fix position of all nodes
  • Release a single locked node?
    Select the node and then right click anywhere on canvas
  • Reset the visualisation layout? Press BACKSPACE key to release all nodes locked in positions
  • Switch to hierarchical layout? Press "H" key to switch to hierarchical layout (press "H" multiple times to try different styles)
  • Stop the wobbling?
    Press SPACE key or double-click anywhere on the canvas (not on a node or an edge)
  • Identify the last expanded node?
    It has a black border (if not selected)
  • Increase the number of neighbour nodes added in a single expansion (double-click)?
    Update the expansion limit by moving the slider in the left pane
  • Move the canvas or zomm in/out?
    Drag&drop the canvas to move and use mouse whell to zoom/in and out
  • Increase/Reduce size of UI elements?
    Press Ctrl++ for increase and Ctrl+- in browser or use mouse wheel while holding down the CTRL key to reduce size of UI elements
  • Preset the name of export file?
    Click on a node of which name you want to use as the name of the exported files (image or JSON)
